# Choice Fracture

A psychological text-based decision game where your past choices silently shape your future.

## Features
- Hidden state system (morality, trust, obedience)
- Multiple endings
- Unreliable narrator concept
- No external libraries

## How to Run
1. Download the project
2. Open `index.html` in a browser
3. Play

## Tech Used
- HTML
- CSS
- JavaScript
